Contact Tracing Capstone

Software designed for tracing contact between employees in the case that an employee tests positive for Covid-19. The program takes a JSON file with employee names and locations throughout the day and uses a clustering machine learning algorithm to determine which employees were within 6 feet of each other within 1 hour. The application can also show scatterplots of different time and location data.

Inventory Management Project

Inventory management system designed for use in an organization with an inventory of parts and products. Has the functionality to add "parts" (either in-house or outsourced), and "products" comprised of one or more parts. Also has the functionality to modify existing parts and products.`

Scheduling Management Project

Appointment Management system designed to help an organization keep track of different types of appointments and meetings. Has the functionality to add new appointments, modify existing appointments, and keep track of logins. Also had the functionality to adjust to different timezones depending on the user machine. At the time of completion the project was connected to a database which stored all data related to appointments, customers, logins, reports, etc. MySQL was used for the database management.
